# Autism Detection System

This project was developed to support early awareness and screening of Autism by providing a simple and accessible computer-vision based system.  
The main motivation behind this project is to help parents and caregivers quickly check whether a child may show possible signs related to Autism, since many people are not aware of Autism and its long-term impact when it is not identified early.  

By offering a quick and easy screening process through a web-based system, the project aims to encourage early attention and timely professional consultation when needed.
